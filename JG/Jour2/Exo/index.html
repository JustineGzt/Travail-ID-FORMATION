Structure d'une page HTML

Les balises de base

Voici les éléments indispensable d'une page HTML :

  1. <!DOCTYPE html> : indique le type de document
  2. <html> : racine du document
  3. <head> : contient les métadonnées
  4. <body> : contient le contenu visible

Les balises de texte

Les balises de texte serbent à organiser et hiérarchiser le contenu :

  • <h1> à <h6> : titres.
  • <p> : paragraphe.
  • <strong> et <em> mise en valeur du texte.
  • <a> : lien hypertexte.

Qu'est ce que le HTML

Le HTML (hyperText Markup langage) est le langage standard pour créer des pages web. Il utilise un système de balises pour Structurer le contenu (titres,paragraphes, images, liens, etc.)
