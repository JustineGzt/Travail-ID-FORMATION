# Profiles Frontend

This project is a frontend application for managing user profiles. It includes a structured layout with reusable components and styles.

## Project Structure

```
profiles-frontend
├── src
│   ├── index.html          # Main HTML document
│   ├── scripts
│   │   └── main.js        # JavaScript functionality
│   ├── styles
│   │   ├── main.scss      # Main SCSS file
│   │   ├── _variables.scss # SCSS variables
│   │   ├── _mixins.scss    # SCSS mixins
│   │   ├── _base.scss      # Base styles
│   │   └── pages
│   │       └── profiles.scss # Styles for profiles page
│   └── components
│       └── profile
│           ├── profile.html # Profile component HTML
│           └── _profile.scss # Styles for profile component
├── package.json            # npm configuration
├── .gitignore              # Files to ignore in version control
└── README.md               # Project documentation
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd profiles-frontend
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the application, open `src/index.html` in your web browser. You can modify the styles in the SCSS files located in the `src/styles` directory and the JavaScript functionality in `src/scripts/main.js`.
